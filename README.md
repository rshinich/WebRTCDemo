# WebRTCDemo

A simple demo with google webrtc.

the signaling is on websocket. 

## Usage

1. clone the code
2. cd WebRTCDemo
3. pod install
4. open the local websocket signaling server
5. run demo
6. deviceA clicked "发送Offer"
7. deviceB clicked "发送Answer"

make sure the "Socket连接状态"、"本地SDP状态"、”远程SDP状态“ are 🟢
